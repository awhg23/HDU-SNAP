# HDU-SNAP

HDU-SNAP 是一个自动做英语单词题的小工具。

你手动登录网页后，它会自动：

- 读取题目
- 把题目发给本地后端
- 选择答案
- 点击“下一题”
- 在第 100 题停住，不会自动提交

这份说明只写给 Windows 同学，而且只保留一条最简单的使用方法。

**快速开始**
先准备两样东西：

- 安装好 `Python 3.10` 或更高版本
- 安装好 `Chrome`

然后按下面做。

1. 下载并解压这个项目。
2. 获取 Deepseek api Key并配置环境变量 DEEPSEEK_API_KEY=你的apiKey，自己搜教程
2. 打开项目文件夹。
3. 在文件夹空白处按住 `Shift` 再点鼠标右键，点击“在此处打开 PowerShell 窗口”。
4. 依次输入下面 4 条命令，每输入一条就按一次回车：

```powershell
py -m venv .venv
.\.venv\Scripts\python.exe -m pip install -U pip
.\.venv\Scripts\python.exe -m pip install -r requirements-lite.txt
.\.venv\Scripts\python.exe main.py
```

5. 程序启动后，输入 `1`，进入正常模式。
6. 打开 Chrome，在地址栏输入 `chrome://extensions/`。
7. 打开右上角“开发者模式”。
8. 点击“加载已解压的扩展程序”。
9. 选择项目里的 `extension` 文件夹。
10. 回到目标网站，手动登录。
11. 进入答题页面，手动点一次“开始”。
12. 后面程序会自动答题，到最后一题会自动停住，等你自己检查。

**你会看到什么**
后端窗口里会持续打印每一题的日志，例如：

```text
[节点校验日志]
第1题: affect
候选项: A. 影响 | B. 效果 | C. 喜爱 | D. 效率
处理方式: 字典匹配
决策结果: A
------------------------
```

答题结束后会显示总统计。

**调试模式怎么用**
如果你想专门记录错题：

1. 启动程序后输入 `0`
2. 跑完一轮后，按提示输入错题信息

推荐这样输入：

```text
12:B 45:D 79:C
```

意思是：

- 第 12 题正确答案是 `B`
- 第 45 题正确答案是 `D`
- 第 79 题正确答案是 `C`

程序会自动：

- 记录错题日志
- 记录系统选错了哪个选项
- 记录正确答案文本
- 自动写入补丁区 [patch_rules.jsonc](./patch_rules.jsonc)

**常见问题**
如果 PowerShell 里提示 `py 不是内部或外部命令`：

- 说明 Python 没装好，或者没有加入系统环境变量

如果扩展加载失败：

- 确认你选择的是 `extension` 文件夹，不是整个项目文件夹

如果程序启动了但网页没有自动答题：

- 确认你已经手动登录
- 确认你已经进入答题页
- 确认你已经手动点过一次“开始”
